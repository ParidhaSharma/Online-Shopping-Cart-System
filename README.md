# Online Shopping Cart System

## Overview
The Online Shopping Cart System is a C++ based project that simulates the basic functionality of an online shopping platform. Users can browse products, add items to a shopping cart, remove products, and complete the checkout process.

The project was developed as part of second-semester coursework and demonstrates the practical application of Object-Oriented Programming concepts in C++.

---
## Features
- Product Catalog Management
- Add Products to Cart
- Remove Products from Cart
- Quantity Management
- Shopping Cart Total Calculation
- Checkout Functionality
- Receipt Generation
- File Storage for Purchase Records
- Menu Driven Interface
